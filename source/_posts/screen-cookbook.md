#screen常用命令

- 在screen外面使用的命令

| 命令                | 解释               |
| :---                | :---             |
| ```screen -ls```    | 查看所有screen会话 |
| ```screen -S XXX``` | 创建署名screen会话 |
| ```screen -r XXX``` | 恢复署名screen会话 |





- 在screen里面使用的命令

| 命令           | 解释               |
| :---           | :---              |
| ```C-a ?```    | 查看帮助           |
| ```C-a S```    | 水平分割屏幕       |
| ```C-a \```    | 垂直分割屏幕       |
| ```C-a c```    | 新建一个窗口       |
| ```C-a n```    | 切换下一个窗口     |
| ```C-a p```    | 切换上一个窗口     |
| ```C-a 0..9``` | 切换到第0..9个窗口 |
| ```C-a k```    | 杀死当前窗口       |
| ```C-a "```    | 查看所有窗口列表   |
| ```C-a d```    | 暂时离开           |
| ```C-a [```    | 进入拷贝回滚模式   |
